# 系統規格書

**系統名稱：** 底稿小幫手
**版本：** 2.0
**文件日期：** 2026-04-15
**類型：** Google Chrome 瀏覽器擴充功能

---

## 一、系統概述

底稿小幫手是一款安裝於 Google Chrome 的瀏覽器擴充功能，提供工作底稿常用文字的集中管理與一鍵複製功能。使用者可將常用的標準文字、條文、用語等預先儲存，工作時直接點擊即可複製到剪貼簿，省去重複輸入或搜尋的時間。

**適用對象：** 需要頻繁輸入固定文字的內部工作人員
**執行環境：** Google Chrome 瀏覽器（桌面版）
**資料儲存位置：** 使用者本機（瀏覽器本地儲存空間）

---

## 二、功能說明

### 2.0 資料夾管理

| 功能 | 說明 |
|------|------|
| 所有文字 | 左側側欄頂部固定項目，點擊可顯示所有資料夾下的所有文字 |
| 新增資料夾 | 可在任一資料夾下建立子資料夾，支援多層巢狀結構 |
| 編輯資料夾 | 修改資料夾名稱 |
| 刪除資料夾 | 刪除資料夾及其所有內容，執行前會顯示確認提示 |
| 排序 | 可將資料夾向上或向下移動調整順序 |
| 隱藏 / 顯示 | 可將指定資料夾隱藏，隱藏後不顯示於一般瀏覽，可透過切換按鈕顯示隱藏項目 |

### 2.2 文字項目管理

| 功能 | 說明 |
|------|------|
| 新增文字 | 建立文字項目，包含名稱（選填）、文字內容、所屬資料夾、標籤；文字必須歸屬於某一資料夾 |
| 編輯文字 | 修改文字項目的所有欄位，包含移動至其他資料夾 |
| 刪除文字 | 刪除文字項目，執行前會顯示確認提示 |
| 一鍵複製 | 點擊文字項目即複製內容至剪貼簿，並顯示「已複製！」提示 |

### 2.3 標籤系統

- 每個文字項目可加上一至多個自訂標籤
- 左側側欄自動彙整所有標籤，點擊標籤可跨資料夾檢視該標籤下的所有文字
- 新增/編輯時可輸入標籤並按 Enter 或逗號確認

### 2.4 最近複製紀錄

- 自動記錄最近複製的文字項目，最多保留 10 筆
- 點擊「最近複製」可快速取用近期常用內容

### 2.5 搜尋

- 提供即時搜尋，依文字名稱、內容、標籤篩選符合結果
- **所有文字模式：** 搜尋範圍跨所有資料夾
- **資料夾模式：** 搜尋範圍限定於該資料夾（含子資料夾）內

### 2.6 匯入 / 匯出

- **匯出：** 將目前所有資料轉換為 JSON 格式，顯示於文字框供複製儲存
- **匯入：** 貼入 JSON 格式資料後，系統會顯示確認視窗，告知現有資料筆數與即將匯入筆數，確認後覆蓋現有資料
- 搭配提供的 `data_template.json` 範本，可讓使用者預先填好資料再匯入

### 2.7 頁面記憶

- 系統自動記憶使用者上次所在的資料夾或頁面
- 重新開啟插件時自動還原至上次瀏覽位置

### 2.8 版本紀錄

- 可查閱各版本更新內容，依功能分類顯示

## 三、技術架構

| 項目 | 說明 |
|------|------|
| 規格版本 | Manifest V3（Chrome 最新擴充功能規格） |

| 資料格式 | JSON 樹狀結構 |
| 外部相依 | 無（不依賴任何第三方函式庫或外部服務） |

## 四、所需權限說明

本擴充功能在安裝時需要以下兩項權限：

| 權限 | 用途 | 說明 |
|------|------|------|
| `storage` | 儲存使用者資料 | 將資料夾與文字內容儲存於本機瀏覽器，關閉瀏覽器後資料不會消失 |
| `clipboardWrite` | 寫入剪貼簿 | 點擊文字項目時，將內容複製到系統剪貼簿 |

**僅申請必要的最小權限，未申請 `tabs`、`cookies`、`history`、`browsingData` 等敏感權限。**

## 五、安全性分析

### 5.1 資料不離開本機

所有資料儲存於 `chrome.storage.local`，**完全儲存在使用者本機的瀏覽器內**，不會傳送至任何外部伺服器或雲端服務。本擴充功能**不進行任何網路請求**。

### 5.2 無法存取網頁內容

本擴充功能未宣告 `content_scripts`，代表它**無法讀取或修改使用者正在瀏覽的任何網頁內容**，操作範圍僅限於擴充功能自身的彈出視窗。

### 5.3 防止惡意輸入（XSS 防護）

程式碼中所有使用者輸入的文字，在顯示前均會經過 HTML 跳脫處理（escaping），將 `<`、`>`、`&`、`"` 等特殊字元轉換為安全字串，**防止跨站腳本攻擊（XSS）**。

### 5.4 匯入資料驗證

匯入功能會驗證輸入是否為合法 JSON 格式，以及是否為陣列結構，格式不正確時會提示錯誤並拒絕載入，**不會執行或注入任何程式碼**。匯入前另有確認視窗顯示現有與匯入資料筆數，防止誤操作。

### 5.5 刪除操作保護

刪除資料夾或文字項目前，均會顯示確認對話框，**防止使用者誤刪資料**。

### 5.6 安全性總結

| 安全項目 | 狀態 |
|----------|------|
| 資料傳輸至外部伺服器 | 無 |
| 讀取瀏覽器歷史紀錄 | 無 |
| 讀取 Cookie 或帳號資訊 | 無 |
| 存取使用者瀏覽的網頁內容 | 無 |
| XSS 防護 | 有 |
| 匯入資料格式驗證 | 有 |
| 匯入操作確認保護 | 有 |
| 刪除操作確認保護 | 有 |
| 使用最新 Manifest V3 規格 | 是 |

## 六、資料備份與還原

使用者可透過「匯入/匯出」功能自行備份資料：

1. **備份：** 點擊匯出，將 JSON 內容複製儲存至本機檔案
2. **還原：** 貼入備份的 JSON 內容後點擊匯入，確認後即可還原
3. **跨裝置移轉：** 將匯出的 JSON 在新裝置上匯入，即可複製整份資料庫
